Repository Signals

  • The project curates dozens of single-topic HTML explainers that cover graphs, dynamic programming, search, and tree algorithms, enabling quick deep links for study sessions.
  • The animatedAlgorithm suite alone ships nine reusable sorting visualizations (Bubble, Bucket, Counting, Heap, Insertion, Merge, Quick, Radix, and Selection) backed by paired HTML/JS modules.
  • An in-house VisualizationLibrary loader stitches shared infrastructure (CustomEvents.js, ObjectManager.js, AnimationMain.js) into every page for consistent playback behaviour.
  • The new Undirected Cycle Detection portrait animation mirrors the BFS/DFS layouts, showing the DFS tree, parent table, and status banner as it flags the first detected loop.
